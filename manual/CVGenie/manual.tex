\section{Overview}

CVGenie is a Game Genie emulator for RackNES with eight parameter banks for
directly controlling RAM addresses on the NES.

\begin{figure}[!htp]
\centering
\includegraphics{CVGenie-Manual}
\end{figure}

\begin{enumerate}
  \item Game selection and display. Used to select the game and load the known addresses.
  \item Parameter banks for selecting controls and manipulating with CV.
\end{enumerate}
